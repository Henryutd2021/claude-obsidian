# Code Availability

## Verbatim statement

> The datasets and codes generated during this study are available at Mendeley Data http://dx.doi.org/10.17632/wngs282m48.1.

## Structural classification

| Check | Classification | Evidence |
|---|---|---|
| Has code DOI? | Yes | The combined statement says codes are available through Mendeley Data DOI: `10.17632/wngs282m48.1`. |
| Has repo? | Yes | Mendeley Data is the named repository. |
| On-request clause? | No | The statement gives a direct DOI and no request condition. |
| Source Data file? | Not inspected | The deposit was not independently opened during this ingest. |
| FAIR-compliance assessment | Partial | The DOI supports findability, but executability, dependencies, license, and documentation were not checked. |

## Lesson for my own CA statements

Lesson: Put code and data under the same DOI only if the repository structure makes code files easy to distinguish from input data, outputs, and documentation.
