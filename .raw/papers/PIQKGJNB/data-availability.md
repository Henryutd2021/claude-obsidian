# Data Availability: PIQKGJNB

Source: Yang et al. 2022, *Nature Energy*, "Data availability" section.

## Verbatim statement

> The data generated during and/or analysed during the current study are available in the main text and supplementary materials. Additional data are available from the corresponding authors on reasonable request. Source data are provided with this paper.

## Structural classification

| Property | Value |
|---|---|
| Statement form | Mixed (paper + SI + on-request + Source Data) |
| Has dataset DOI? | No |
| Has repository deposit? | No (no Zenodo / Figshare / Dryad / Mendeley Data / institutional repo) |
| "On reasonable request" clause? | Yes (for additional data beyond paper + SI + Source Data) |
| Source Data file provided? | Yes (publisher-side, not in this Zotero record) |
| FAIR-compliant by 2026 Nature standards? | Partially (Source Data file present; persistent identifier missing for the underlying scenario data) |

## Lesson for my own DA statements

Evidence: the strongest DA element here is "Source Data are provided with this paper" — a publisher-managed file at a stable URL with the article DOI.

Inference: the weakest element is "additional data available from the corresponding authors on reasonable request" — by 2026 Nature now treats this as a yellow flag for any computed scenario data that is not in Source Data or SI.

Lesson: when I write DA for my own manuscripts, the goal is to make the "on reasonable request" clause genuinely empty. Deposit scenario outputs at a Zenodo DOI even when not required.
