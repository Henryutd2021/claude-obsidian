# Data Availability

## Verbatim statement

> The formatted data we used for industrial thermal demands are available via Zenodo at https://doi.org/10.5281/zenodo.11176520 (ref. 61).

## Structural classification

| Check | Status | Note |
|---|---|---|
| Has dataset DOI? | Yes | DOI: `10.5281/zenodo.11176520` |
| Has repository or archive? | Yes | Zenodo archive named in the statement and reference 61. |
| Has on-request clause? | No | The statement gives a public archive DOI. |
| Has Source Data file? | No local Zotero Source Data child | The article says extended data and supplementary material are online, but Zotero has only the main PDF child. |
| FAIR-compliance assessment | Partial | The DOI supports findability and citation. This ingest did not inspect metadata completeness, license, file formats or reuse instructions inside Zenodo. |

## Lesson for my own DA statements

Lesson: Put the cleaned analysis data in a citable archive with a DOI, then name exactly which data layer the archive contains. If raw data are excluded or transformed, say so directly in the statement or a short repository README.
