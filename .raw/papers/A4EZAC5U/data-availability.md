# Data Availability

## Verbatim statement

> All data generated or analysed during this study are included in the article and its Supplementary Information and Data files. Source data are provided with this paper.

## Structural classification

| Question | Classification | Evidence |
|---|---|---|
| Has dataset DOI? | Partial | The data statement does not give a dataset DOI, but the Nature page provides Supplementary Data and Source Data files. |
| Has repository? | Yes for code, not a separate data repository in the DA statement | The DA statement points to article, Supplementary Information, Data files and Source Data. |
| On-request clause? | No | No request-only language appears in the DA statement. |
| Source Data file? | Yes | The Nature page lists Source Data Fig. 1 through Source Data Fig. 6. |
| FAIR-compliance assessment | Medium | Files are public through the publisher page, but the DA statement itself does not name a persistent dataset DOI for the data files. |

## Lesson for my own DA statements

Lesson: Name the exact data objects and give persistent identifiers where possible. A stronger version for Henry's work would say which files reproduce which figures and include a DOI for the data package, not only a publisher-hosted Source Data pointer.
