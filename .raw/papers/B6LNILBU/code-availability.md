# Code Availability

## Verbatim quote

> All of the code used to process the data and produce all analyses and figures is publicly available at https://doi.org/10.5281/zenodo.6989075.

## Structural classification

| Check | Classification | Evidence |
|---|---|---|
| Has dataset DOI? | Yes | `https://doi.org/10.5281/zenodo.6989075` |
| Has repo? | Yes | DOI points to the public code archive. |
| On-request clause? | No | The statement says all code is publicly available. |
| Source Data file? | Yes | Data and code share the Zenodo archive path. |
| FAIR-compliance assessment | Medium-high | Public DOI and complete-code claim are strong. Independent link and file-structure verification were deferred. |

## Lesson for my own CA statements

Lesson: Use one direct sentence that states scope and archive location: all processing code, analysis code and figure code, followed by a DOI or repository URL.
