# Data availability

## Verbatim statement

> We used data from IEA world energy balances53 for wind and solar power generation, data from refs. 53,78–87 for the independent variables used in the statistical analyses (see Supplementary Note 5 for details) and data from Huppmann et al.46 to calculate the growth rates in scenarios as reported in the Supplementary Data. Source data are provided with this paper.

## Structural classification

| Question | Classification | Evidence |
|---|---|---|
| Has dataset DOI? | Partial | The statement cites source datasets and the Supplementary Data, but it does not give a single dataset DOI for the paper's processed data package. |
| Has repository or data portal? | Yes | The paper references IEA world energy balances, IPCC scenario data, OpenEI, World Bank, IRENA, Global Solar Atlas and Global Wind Atlas URLs. |
| Has on-request clause? | No | The statement says source data are provided with the paper. |
| Source Data file in Zotero? | No | Zotero children list only the main PDF attachment `SBU9QH66`. |
| FAIR-compliance assessment | Partial | Sources and source data are named, but the statement does not itself specify file inventory, license, variable dictionary, version tag or reproducibility order. |

## Lesson for my own DA statements

Lesson: State the processed-data package as a citable object, then list each external source, file type, version date and mapping from source files to figures so readers can reproduce the data chain without reverse-engineering the Methods.
