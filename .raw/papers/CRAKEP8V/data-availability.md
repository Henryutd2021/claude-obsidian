# Data availability

## Verbatim statement

> The data that support the findings of this study are provided in Supplementary Data 1. The vectorized rooftop area data for 90 cities in China are available at https://doi.org/10.11888/Geogra.tpdc.271702. Other data sources that are free to use are provided in the supplementary materials (Supplementary Table 14).

## Lesson for my own DA statements

Lesson: Name the exact supplementary data file and the external dataset DOI in the same statement, then add license, version and access date if the dataset will be reused by other researchers.
