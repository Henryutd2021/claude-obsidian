| Check | Status | Note |

| Has repo? | Yes | The article points to an open GCAM repository archive. |

| FAIR assessment | Good for L1 ingest | The model archive is citable, but this ingest did not inspect whether all scenario input files are sufficient to rerun the paper. |

## Lesson for my own CA statements

Lesson: Cite the exact model archive and version, then add project-specific scenario scripts and configuration files. For Henry's work, a model DOI alone is not enough if the paper uses custom scenario branches.
