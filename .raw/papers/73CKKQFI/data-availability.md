# Data Availability

## Verbatim Statement

> "The underlying data for all figures in the main text and in the supplemental information is available in an open repository at Zenodo (https://doi.org/10.5281/zenodo.10934198)."

## Structural Classification

| Check | Status | Note |
|---|---|---|
| Has dataset DOI? | Yes | Zenodo DOI: `10.5281/zenodo.10934198`. |
| Has repo? | Yes | The data statement points to an open Zenodo record for figure data. |
| On-request clause? | No | The statement does not use a request-only gate for figure data. |
| Source Data file? | Yes, external | Data are external at Zenodo, not present as a Zotero attachment in this ingest. |
| FAIR assessment | Good for L1 ingest | DOI-backed figure data are provided, but the repository contents were not independently inspected during this ingest. |

## Lesson for my own DA statements

Lesson: Put all figure data in a DOI-backed repository and name the repository in the article's resource statement. For Henry's work, add a short manifest that maps each figure panel to a file name so reviewers can audit the path from model output to figure.
