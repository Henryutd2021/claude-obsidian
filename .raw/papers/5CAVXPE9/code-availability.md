# Code availability

## Verbatim statement

> The ResFrac reservoir simulation code is commercial software developed by the ResFrac Corporation. The GenX electricity system capacity expansion model is open source and available via GitHub at https://github.com/GenXProject/GenX. Source code for the modified version of GenX used in this work is available via Zenodo at https://doi.org/10.5281/zenodo.7023225 (ref. 49).

## Structural classification

| Question | Classification | Evidence |
|---|---|---|
| Has dataset DOI? | Yes | The modified GenX source is available through Zenodo DOI `10.5281/zenodo.7023225`. |
| Has repo? | Yes | The public GenX repository is `https://github.com/GenXProject/GenX`. |
| On-request clause? | No for code | The code statement does not make code availability request-only, but ResFrac is commercial software. |
| Source Data file? | Not applicable | This statement concerns model code, not source data files. |
| FAIR-compliance assessment | Partial | GenX and modified GenX are reachable, while the reservoir simulation engine is commercial and not fully reproducible from open code alone. |

## Lesson for my own CA statements

Lesson: If part of the workflow uses closed software, name it directly and then make the open part of the pipeline as reproducible as possible with a repository or archival DOI.
